# html-to-do-list-# 📝 Professional Task Manager

A modern and responsive Task Management web application built using **HTML, CSS, and JavaScript**. The application helps users organize daily tasks, assign categories, set due dates, track completion status, and manage productivity with live analytics.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Mark tasks as completed
- 🗑️ Delete tasks
- 📅 Set due dates
- 🏷️ Categorize tasks (Work, Personal, Urgent)
- 🔍 Search tasks instantly
- 🎯 Filter tasks by category
- 📊 Live analytics dashboard
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
  - Progress Percentage
- 💾 Local Storage support (tasks remain after refreshing the browser)
- 📱 Responsive user interface
- 🌙 Modern dark-themed dashboard

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
